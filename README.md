# YouTube Content Creation Agent — Enterprise Edition 🚀

Welcome to the production-ready AI SaaS version of the YouTube Content Creation Agent. This project has been transformed with full DevOps, DevSecOps, and LLMOps practices.

## 📁 Project Structure
- `frontend/`: Streamlit dashboard
- `backend/`: Flask API & core logic
- `ai/`: Prompt templates & AI services
- `docs/`: Full setup and deployment guides
- `docker/`: Docker configuration
- `k8s-simple/`: Kubernetes manifests for KIND and EKS
- `.github/workflows/`: CI/CD pipelines
